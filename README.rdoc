The Pantry

The Pantry is an online personal recipe book that allows user to create and save thier favorite recipes. By creating an account, the user can also edit and delete their recipes.

The Pantry is deployed on Heroku

Technologies include:

Ruby 2.2

Rails 4.2

PostgreSQL 9.4

Twitter Bootstrap 3.3

SASS 3.4

bootstrap-generators 3.3
